# Planning Navigator

An interactive questionnaire tool for planning permission assessment. This tool helps users understand whether planning permission is likely to be required or whether Permitted Development may apply.

## Features

- Interactive questionnaire flow
- Dynamic question routing based on answers
- Accordion-style nested questions for detailed checks
- Real-time guidance and feedback
- Comprehensive result page with recommendations
- Responsive design for all devices

## Usage

1. Open the application in your browser
2. Answer the questions about your property, project, and site
3. Receive guidance and recommendations based on your answers
4. Review the comprehensive result page with next steps

## Technical Details

- Pure HTML, CSS, and JavaScript (no external dependencies)
- Single-page application
- Fully self-contained
- Compatible with all modern browsers
